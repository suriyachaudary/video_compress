# video_compress

Download and setup [Eigen](https://libeigen.gitlab.io/eigen/docs-5.0/GettingStarted.html)

Compile code with

``g++ test_image_compress.cpp -o test_image_compression `pkg-config --cflags --libs opencv4` ``

Run code with 

``./test_image_compression [path_to_image]``
